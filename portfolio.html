Fighting Game

JavaScript

Side project made with tutorial at: https://youtu.be/vyqbNFMDRGQ

A two-player fighting game that uses keyboard controls to control the characters. Canvas was used within JS to draw and manipulate the sprites along the screen, a gravity and animation system was coded in as well as a very basic physis engine. The UI was implemented using HTML ontop of the canvas and uses animations downloaded online.

You can play the game here: https://faiza-maghrabi.github.io/fighting-game/

This project required the use of JS's canvas, many sprites and multiple classes to deal with methods called multiple times.

Monster Clicker

Java

OOP Miniproject

A small game that consisted of a few buttons for the user to click. When the user presses an attack button, damage is done to the monster - based on the weapon currently equipted. When the monster is defeated, the player gets money and this can be used to get new weapons that deal more damage and lets the user gain more money quicker.

This project includes polymorphism, inheritance, read/write methods, utilisation of multiple spirites, and interface implementation.

Secret Traps and Passages

Java

Miniproject for Year 1 Semester A's Procedural Programming

A board-game-like program for multiple people. Each player rolls a random die and moves across the board, they can also answer questions and this may alter their position. The first one to the end of the board wins. The game can be ended after every turn and data is saved and read from playerDetails.txt

This project required the use of multiple classes, a gameloop as well as read & write functions.

Text Compressor

C#

Part of my A-level Project

A compressor that takes multiple lines of text, creates a dictionary based on the number of unique words available. This dictionary is then used to create an RLE style compressed file.

This project includes encode and decode methods as well as read and write to file methods
